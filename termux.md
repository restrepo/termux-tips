# Termux
Termux is an Android terminal emulator and Linux environment app
## Install
 See the official help in [Termux Wiki
](https://wiki.termux.com/wiki/Installing_from_F-Droid)

## Tips
Open Termux and 
### Update and upgrade
```bash
termux-change-repo
pkg upgrade
```
### Activate storage 
Where android user files are
```bash
termux-setup-storage
pkg i git 
```
